# Data Analytics, Data Science, Machine Learning & AI Roadmap

This roadmap shows how the course topics connect.

## 1. Python Foundation

Learn:

- Variables
- Data types
- Conditions
- Loops
- Functions
- Lists, tuples, sets, and dictionaries
- File handling

## 2. Data Analytics

Learn:

- Data collection
- Data cleaning
- Excel basics
- CSV files
- Pandas
- NumPy
- Charts and dashboards

## 3. Data Science

Learn:

- Exploratory data analysis
- Statistics basics
- Data visualization
- Feature selection
- Model evaluation basics

## 4. Machine Learning

Learn:

- Supervised learning
- Unsupervised learning
- Regression
- Classification
- Clustering
- Model training and testing

## 5. Artificial Intelligence

Learn:

- What AI is
- How machine learning is used in AI
- Prompting basics
- AI tools for learning and coding
- Responsible AI usage

## Suggested Learning Rule

Do not rush. First understand Python clearly, then move step by step into data analytics, data science, machine learning, and AI.
