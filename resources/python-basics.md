# Python Basics

Python is a beginner-friendly programming language used in software development, data analytics, data science, machine learning, and AI.

## Print Output

```python
print("Hello, Python")
```

## Variables

```python
name = "Rahul"
age = 20
marks = 85.5
is_student = True
```

## Common Data Types

```python
student_name = "Anjali"   # string
student_age = 19          # integer
student_marks = 91.5      # float
is_passed = True          # boolean
```

## List

```python
numbers = [10, 20, 30, 40, 50]
print(numbers)
```

## Dictionary

```python
student = {
    "name": "Priya",
    "course": "Python and AI",
    "city": "Delhi"
}

print(student)
```

## Function

```python
def greet(name):
    print("Hello", name)

greet("Rahul")
```

## Practice Tips

- Write code daily.
- Run your code before submitting.
- Read error messages carefully.
- Keep variable names meaningful.
- Ask questions when you are stuck.
